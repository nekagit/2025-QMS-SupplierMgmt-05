# Supplier Management - 2025-QMS-SupplierMgmt-05

A Quality Management System application for managing and evaluating supplier performance.

## Features

- Register suppliers
- Supplier rating system (A, B, C, D)
- Category classification
- Status tracking
- Audit date management
- Supplier listing and deletion

## Tech Stack

- **HTMX** - Dynamic HTML interactions
- **SCSS** - Styling
- **TypeScript** - Type-safe JavaScript
- **Express.js** - Server framework

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Usage

1. Start the server: `npm run dev`
2. Open your browser to `http://localhost:3005`
3. Register suppliers using the form
4. View and manage suppliers in the list
